## 🛫 What this is

ACBuy Spreadsheet is framed as a **buyer control tower** rather than a plain link list. The goal is to help you route finds through a clean decision flow:

```text
Candidate find
   ↓
Route radar scan
   ↓
Sizing runway check
   ↓
QC flight log review
   ↓
Parcel runway decision
   ↓
Ship only after clearance
```

The site is built for shoppers who want fewer random guesses and more confident decisions: clearer category routes, stronger QC questions, better sizing checks, and smarter parcel planning.

## ✅ Pre-flight checklist

Before approving international shipping, run the item through this checklist:

- [ ] The route has a clear reason to exist in the haul.
- [ ] Listing photos show the details that matter for the category.
- [ ] The sizing runway is clear enough to avoid fit guessing.
- [ ] Warehouse QC photos prove the key details.
- [ ] Parcel weight and volume still make sense.
- [ ] A backup route exists if QC fails.

---

## 🧭 How to use the tower

1. **Pick a radar sector** — shoes, hoodies, jackets, accessories, or another category.
2. **Compare multiple candidates** — do not treat the first good-looking listing as final.
3. **Submit the strongest route** — choose the candidate with the best evidence.
4. **Review QC photos like proof** — warehouse arrival is not approval.
5. **Clear the parcel runway** — check weight, box removal, line choice, and timing.
6. **Approve only when the route is clear** — otherwise measure, exchange, or return.
